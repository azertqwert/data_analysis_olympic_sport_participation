# Sport Participation and Olympic Success

This repository contains a university project for the course
**Open Access and Digital Ethics**.

The project explores the relationship between sport participation rates
and Olympic success across countries using open datasets.

## Objectives
- Explore inequality and access to sport
- Combine datasets with different licenses (mashup)
- Respect legal, ethical, and privacy requirements
- Publish open results with metadata and visualizations

## Contents
- Data documentation
- KNIME workflow
- Mashup dataset
- Visualizations
- DCAT-AP metadata
- One-page website
